# 🚀 Jared Kong — Portfolio

Welcome to the source for [**jaredkong.com**](https://jaredkong.com) — my personal portfolio. I'm Jared Kong, a Full-Stack & AI Agent Engineer based in Singapore, currently in Year 4 Computer Science at NTU (graduating Dec 2027) and building agentic AI products as a software engineer at Voltade. This site is where I show off the work, the wins, and the weird-fun side of how I build things on the web. ✨

## 🔥 Features

### 🎹 **3D Interactive Keyboard**
- A custom **Spline** scene where each keycap is a skill — hover to reveal titles and descriptions.
- Smooth, responsive 3D interactions designed to be playful but informative.

### ✨ **Slick Motion**
- Powered by **GSAP** and **Framer Motion** for buttery scroll, hover, and reveal animations.
- Motion serves the storytelling — never decorative for its own sake.

### 🌌 **Space-Themed Design**
- Shooting stars across a dark cosmic backdrop set the mood.
- Just the right amount of "futuristic" without sliding into cliché.

### 📱 **Responsive by Default**
- Looks and feels right from a phone in the lift to a 27" desktop.

### 📬 **Working Contact Form**
- Powered by **Resend** — messages land directly in my inbox.

## 🛠️ Tech Stack

- **Framework:** Next.js 14, React 18, TypeScript
- **Styling:** Tailwind CSS, shadcn/ui, Aceternity UI
- **3D & Animation:** Spline, GSAP, Framer Motion
- **Email:** Resend

## 🌟 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Jxredkong/3D-interactive-portfolio.git
   cd 3D-interactive-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables — create `.env.local` in the project root:
   ```bash
   RESEND_API_KEY=your_resend_api_key_here
   ```

4. Start the dev server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000).

## 🚀 Deployment

Deployed on **Vercel** for instant global delivery. To replicate:

1. Push this repo to your own GitHub account.
2. Import the project in Vercel.
3. Under **Project Settings → Environment Variables**, add:
   - `RESEND_API_KEY` — your [Resend](https://resend.com) API key.
4. Deploy. That's it. 🎉

> ⚠️ Never commit your `.env.local` or any API keys.
